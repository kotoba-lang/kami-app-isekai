スライムハント — Slime Hunt

Real-time dodge/collect arena — WASD/arrows to move. Dodge the 3 slimes (3 lives), collect all 8 glowing kotodama orbs to win. Compiled via kotoba.engine-clj to real WASM, run through kami.host, rendered via kami.scene2d + kami.webgl (GPU-instanced quads, WebGL2) — the same harness shape gftdcojp/network-isekai's "drive" game already runs in production, adapted for this repo.
